void loadbalancer::main_loop ( )

This runs the simulation and assigns each server with its request onces it's avaialble.

After beginning the clock, the function runs an endless loop until either the clock length has been reached or the requestqeue has become empty. The first condition is to ensure the loop is ends within a reasonable amount of time of the inputted time. The second condition is to ensure every request is handled before ending the loadbalancer. After the loop has begun, we randomly generate a number of requests to add to the queue, representing actual traffic. After which, the function iterates through all the servers and checks if they're available for a new request, if not then continue, otherwise assign the server the request and move to the next server.
