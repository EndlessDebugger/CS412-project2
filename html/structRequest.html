Detailed Description

A struct holding the incoming and outgoing ip adresses, the time length of a simulated request.

This object is used for a webserver object to hold onto while running to simulate a server handling requests.

string Request::gen_ip ( )
inline

IPV4 Address generation.

When called it will call rand()%256 (0 to 255) to generate one part of an IPV4 address. Thus it's structure is: 'rand()%256'.'rand()%256'.'rand()%256'.'rand()%256' Each part is converted and merged into one string.

Returns
a String parameter holding a randomized IPV4 Address.
