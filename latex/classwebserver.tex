{\footnotesize\ttfamily bool Web\+Server\+::status (\begin{DoxyParamCaption}\item[{int}]{curr\+\_\+time }\end{DoxyParamCaption})\hspace{0.3cm}{\ttfamily [inline]}}



Indicates if the server is available to take on a new request. 

This function checks two conditions to make the determine if the server is unoccupied, otherwise assume it\textquotesingle{}s still processing a \mbox{\hyperlink{structRequest}{Request}}. occupied is set True via \mbox{\hyperlink{classWebServer_a1f688babb28b5e6af7170852351ceaf7}{set\+\_\+job()}}. First condition is if the current request has a time length of 0, this would only be possible if was created via the default constructor thus the server has previously completed a request and is ready for a new one. The second condition checks on if the current request has been completed. This is accomplished via time arithmetic, in which it calls endjob(curr\+\_\+time).
